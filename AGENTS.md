# agents.md

## 共通ルール

- やり取りはすべて **日本語** で行うこと
- 作業を開始する前に **Todo.md** に「やるべきこと」を書き出すこと
  - 作業完了後には **やったこと** を追記すること
- 変更はなるべく細かく **git commit** で管理すること
  - コミットメッセージは簡潔に日本語で書くこと
  - `git add -p` を活用して関連する変更だけをまとめること
- **テストがすべてグリーン** であることを確認してからコミットすること
- コメントは **すべて日本語** で記載すること
- 多言語対応（日本語／英語）を前提に実装すること

## プロジェクト概要

Obsidian 用のプラグイン **「obsidian-file-navigator」** を開発する。  
機能概要は以下。

### 基本機能

- **開いているファイルを基準**に、一定のルールに従ってファイルを移動する
- ルールは以下を含む：
  - フィルター条件
    - タグ（指定されたタグを持つファイル）
    - フォルダ（指定されたフォルダ直下のファイル）
    - ※ 1 つのルールにつきタグまたはフォルダを指定
  - ソート条件
    - 作成日
    - 更新日
    - ファイル名
    - frontmatter の特定キー（数値／文字／日付）

### グループ

- 複数のルールをまとめた単位（ショートカットコマンドごと）
- グループごとにショートカットを持つ：
  - 前のファイルへ
  - 次のファイルへ
  - 最新のファイルへ
- ルールが複数ある場合は **優先順位** に従って適用する
